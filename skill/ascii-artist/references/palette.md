# ASCII Art Character Palette

## Density Ramp (light → dark)
Use these sequences for shading and gradients. Pick one ramp and stay consistent within a piece.

```
Standard:    .:-=+*#%@
Extended:    .'`^",:;Il!i><~+_-?][}{1)(|/tfjrxnuvczXYUJCLQ0OZmwqpdbkhao*#MW&8%B@$
Blocks:      ░▒▓█
Minimal:     .:#
Dots:        ·∘○◎●◉
```

## Gradients & Fills
```
Horizontal gradient:  ░░░░▒▒▒▒▓▓▓▓████
Vertical gradient:    Use lighter chars at top, heavier at bottom
Stippled fill:        .·:·.·:·.·:·
Water/liquid:         ≋≈∿～〜∼
Sand/grain:           .:·.·:.:·.
Grass:                ⌇⌇⌇  or  ^^^  or  ,,,',,,
Snow/ice:             ·  · · · ·  ·
```

## Shading Techniques

### Halftone (simulate grayscale)
```
100%:  ████████
 75%:  ▓▓▓▓▓▓▓▓
 50%:  ▒▒▒▒▒▒▒▒
 25%:  ░░░░░░░░
  0%:  (space)
```

### Cross-hatch
```
Light:    . · . · .
Medium:   :.:.:.:.
Heavy:    #:#:#:#:
```

### Shadow casting
Place ░ or ▒ to the right and below solid objects to create shadows:
```
████████
████████░
████████░
  ░░░░░░░
```

The skill-enhanced version is a dramatic leap: bamboo fence frame (╤╤╤), raked sand patterns using .:·.·:. texture, ▓█ shaded rocks with ░ shadows beneath them, a stone lantern built from box characters (┌┘└┐├───┤), bamboo trees with ╱╲ branching, koi fish (><(((°> and <°)))><) swimming in ≋ water, stepping stones (◯), a wooden bridge (═║), ⌇ moss ground, and a legend. This is where the skill shines most — transforming a simple prompt into a rich atmospheric scene with proper layering (sky → structures → sand → water → ground).
